# webshop

Sklep internetowy:
Backend: Java Spring Boot
Frontend: Angular

Dodatkowe technologie: Bootstrap, Lombok

Opis:
Aplikacja internetowa - ze szczególnym nastawieniem na panel administratora.
Przewidywane funkcje: 
-Rejestracja użytkownika, Logowanie
-Dodanie produktu do magazynu, edycja, usówanie
-Lista zmówień
-Utworzenie koszyka i dodawanie produktów do koszyka.
-Wysłanie koszyka

Relacyjna baza danych - w fazie produkcujnej wykorzystuję H2.
